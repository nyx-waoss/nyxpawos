❔ Ayuda

Solucion de problemas al iniciar sesion:

Olvide mi contraseña

Si no recuerdas tu contraseña intenta lo siguiente:

  • Haz clic aqui para restablecer tu contraseña.
  • Verifica si estas ingresando tu contraseña correctamente (Mayusculas, Minusculas, Espacios).
  • Intenta iniciar sesion mas tarde, el sistema puede estar en una actualizacion.
  • Investiga si otras personas tienen el mismo problema.

Error interno del sistema

Si aparece un error al intentar iniciar sesion:

  • Reinicia el sistema y vuelve a intentarlo.
  • Presiona ctrl + shift + R para borrar la caché.
  • Cierra y vuelve a abrir el navegador.
  • Actualiza el sistema desde el launcher.

Problemas de conexion

Si el sistema no carga o dura demaciado:

  • Actualiza el sistema desde el launcher.
  • Verifica tu conexion a internet.
  • Intenta conectarte a otra red.
  • Desactiva cualquier VPN o extenciones que puedan modificar la conexion.

Cuenta bloqueada o deshabilitada

Si el sistema indica que tu cuenta no esta disponible:

  • Espera unos minutos e intentalo de nuevo.
  • Investiga si otras personas tienen el mismo problema
  • Intenta restablecer tu contraseña.
  • Actualiza el sistema desde el launcher

Si el problema continúa, por favor contacte a soporte técnico y proporcione detalles sobre el error.
