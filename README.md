NyxPawOS is an open source operating system that runs entirely in your browser — no installation, no backend. It features a full desktop environment with 25+ apps, a custom file system powered by IndexedDB, virtual machine support, and a conversational assistant with personality, all built with vanilla JavaScript.

## ✨ Top Features
**NyxPaw Workspace** — A full productivity suite including a document editor, spreadsheets, slides, an AI-powered notebook, notes, calculator, databases, and more.

**NeptuneFS** — A custom file system built on IndexedDB. Supports text files, images, PDFs, folders, presentations, and videos — all stored locally in your browser.

**VirtualEnv** — A virtual machine manager inspired by VirtualBox. Run other web-based operating systems inside NyxPawOS, each with its own name and hardware configuration.

**NyxPaw Store** — An app store built into the OS to browse and install additional apps.

**Toybox** — A collection of absurd, useless, and absolutely fun system tweaks: mouse trails, exaggerated animations, blur effects, and more. Inspired by Tesla's easter egg menu.

**Nekiri** — A conversational assistant with a lot of personality. Nekiri won't automate your life, but will talk to you, listen, and open apps when you ask.
