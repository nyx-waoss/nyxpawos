📦 Suite de Productividad NyxPaw

  • Se incorporan Workspace, Docs, Slides y Sheets, diseñadas para gestionar trabajo y tareas directamente desde el sistema.
  • Mejoras en el soporte para distintos formatos de documentos, permitiendo abrirlos y editarlos sin errores de lectura.
  • Se añade un visor integrado que permite abrir y realizar ediciones básicas en documentos PDF de forma nativa.
  • Nueva aplicación de Contactos para gestionar personas, información y accesos de forma centralizada.

🎨 Interfaz y Experiencia de Usuario (UX)

  • Actualización visual con bordes redondeados, nuevas animaciones y fondos de pantalla para un entorno más moderno.
  • Se centralizan los ajustes del sistema en un solo panel para que configurar el equipo sea más rápido y sencillo.
  • Nuevos iconos en aplicaciones y menús para mejorar la identificación visual de las funciones.
  • Uso de Web Components y menús desplegables mejorados que ofrecen una respuesta más ágil al interactuar.
  • Se añade un nuevo control en Toybox que permite adaptar el radio de los bordes de las ventanas según el dispositivo o estilo visual (Default, MacBook Neo, Custom, entre otros).

⚙️ Núcleo y Rendimiento del Sistema

  • Mejoras en el sistema para que consuma menos recursos (RAM/CPU) y se sienta más ligero al abrir aplicaciones.
  • Actualización de los protocolos internos para proteger mejor el acceso a tus datos personales.
  • Nuevo sistema de detección de crashes. Pantalla de crash y Menu de emergencia rediseñados: ahora puedes presionar Shift + F para obtener diagnósticos detallados.
  • Implementación de un nuevo módulo de bloqueo de pantalla para proteger el acceso al sistema.
  • Nuevo sistema de confianza de archivos que permite identificar, validar y restringir la ejecución de contenido potencialmente inseguro.

💻 Herramientas Avanzadas y Desarrollo

  • Mayor estabilidad al usar comandos y adición de nuevas funciones para usuarios que prefieren la terminal.
  • Mejora en el modelo de IA residente, logrando respuestas más naturales y una mejor comprensión de las peticiones.
  • Nueva aplicación para acceder a contenido de video (requiere configuración de API Key).
  • Expansión del soporte para múltiples idiomas.
  • Nuevo sistema de ejecución automática al iniciar sesión, permitiendo gestionar aplicaciones y scripts de arranque.
  • Soporte para nuevos scripts JavaScript con extensión .npfr, integrados con el sistema para automatización y personalización avanzada.

Solucion de problemas

Si el boton de inicio no funciona:

  • Asegurate de activar Javascript: Como activar Javascript
  • Asegurate de permitir los PopUp en la configuracion del sitio

Si aparece una pantalla en negro con un texto y no hay ninguna barra de progreso:

  • Asegurate de activar Javascript: Como activar Javascript
  • Verifica que tengas la ultima version del sistema
  • Investiga si alguien mas tiene el mismo problema
  • Reinstala el sistema operativo y no modifiques los archivos si no sabes lo que haces

Si el sistema de archivos no funciona:

  • Asegurate de activar Javascript: Como activar Javascript
  • Asegurate que nada este bloqueando el acceso a local storage o indexed DB
